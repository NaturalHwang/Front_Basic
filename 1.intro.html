HTML 학습하기

h태그는 제목을 의미하고 위아래로 약간의 여백이 자동 삽입

검색 엔진에서 h태그는 중요한 검색 키워드로 사용, h1~h6까지 존재

p태그는 단락을 의미하고, 줄바꿈이 되면서 태그 위아래로 약간의 여백이 삽입

html 태그 내에서는 엔터를 치더라도 줄바꿈이 되지 않고,
br태그를 써야한다(종료 태그 없음). 또한 웹 브라우저에서는 여러번 띄어쓰기를 하더라도 한번만 띄어쓰기가 됨에 유의.

텍스트의 형식화

굵게와 강조된 굵게가 사용되고, 이텔릭체와 강조된 이텔릭체가 있습니다. 삭제된 텍스트를 표현할 때 del 태그 사용

html에서의 엔티티(특수 기호) 사용

여러개의 공백을 넣고 싶을 때에는     nbsp를 사용한다.

단락을 표시하는 태그는 <p>태그 이다.

div태그는 특별한 기능이 없고, 다른 HTML 요소들을 하나로 묶어 디자인 요소를 적용하는데 주로 사용된다. 줄바꿈됨
span 태그는 특별한 기능이 없고, 마찬가지로 CSS 적용을 위해 사용. 줄바꿈이 없다.
